== Description ==

Mekanews Lite is a responsive, stylish, user-friendly and SEO-friendly theme that focus on news portal and personal blog.

== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Type MekaNews Lite in search box. Click Install Now on the theme.
3. Click Activate to use your new theme right away.
